# 📚 Library Management System using FastAPI

## 📌 Project Description

This project is a Library Management Backend System developed using **FastAPI**.  
It provides APIs to manage books, borrowing, returning, searching, sorting, and pagination.

The system simulates real-world library operations such as borrowing books, maintaining records, and managing waiting queues.

---

## 🚀 Features Implemented

This project includes the following FastAPI features:

- GET APIs
- POST APIs with Pydantic validation
- PUT APIs (Update Book)
- DELETE APIs (Delete Book)
- Helper Functions
- Borrow & Return Workflow
- Queue Management System
- Search Functionality
- Sorting Functionality
- Pagination
- Combined Search + Sort + Pagination
- Data Validation using Pydantic
- Swagger API Testing

---

## 🛠 Technologies Used

- Python 3.x
- FastAPI
- Uvicorn
- Pydantic
